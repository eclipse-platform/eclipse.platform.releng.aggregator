Java Element Filters Dialog

This dialog lets you define Java element filters for the Package Explorer view and other Java views.

Option

Description

Default

Name filter patterns

If enabled, a comma separated list of patterns can be specified additionally.

Off

Select the elements to exclude from the view

List of pre-defined filters which can be enabled.

.* resources,
Empty library containers
Empty parent packages,
Import declarations,
Inner class files,
Package declarations,
Synthetic members

The Filter description field displays the description for the currently selected filter.
