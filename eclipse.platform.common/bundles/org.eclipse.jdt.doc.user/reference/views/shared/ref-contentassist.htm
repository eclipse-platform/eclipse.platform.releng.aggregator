Content Assist

Select the Content Assist command [ ] to open the Content Assist popup dialog. You can also use the keyboard shortcut Ctrl+Spacebar within the view.

The resulting dialog provides you with context sensitive coding help by making available a listing of all applicable java code elements for the location content assist was activated.

This command applies to:

  • Java Editor
  • Debug Shell View
  • Detail Pane (in the Expressions View and Variables View)
