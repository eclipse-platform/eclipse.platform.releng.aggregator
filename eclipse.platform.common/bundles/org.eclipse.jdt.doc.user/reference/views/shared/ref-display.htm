Display

Select the Display command [ ] to have the selected statement evaluated and the results displayed inline in the Debug Shell View or in a popup dialog (that can be moved to the Display View).

This command applies to:

  • Java Editor
  • Debug Shell View
  • Detail Pane (in the Expressions View and Variables View)
  • Run Menu
