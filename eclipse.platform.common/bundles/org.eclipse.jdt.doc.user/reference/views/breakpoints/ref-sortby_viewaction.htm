Sort Breakpoints By

Select the Sort By view menu item to sort the breakpoints in the Breakpoints View in one of a choice of predefined orderings.

The orderings are as follows:

  • Name - breakpoints are sorted by name
  • Creation Time - breakpoints are sorted by Creation Time
