Toggle Watchpoint

Select the Toggle Watchpoint command [ ] to create a new watchpoint on the selected field in either the editor or the Outline View.

Note that you cannot set a watchpoint on a final or a static final constant field; if you try to do so you will get a line breakpoint instead. This cannot be implemented for final variables with compile-time constant values because Java compiles the constant value into class files and leaves no trace of a field access at points where the final field is read.
