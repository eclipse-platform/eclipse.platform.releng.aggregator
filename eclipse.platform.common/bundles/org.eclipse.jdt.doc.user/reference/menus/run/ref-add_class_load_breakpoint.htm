Add Class Load Breakpoint

Select the Add Class Load Breakpoint command [ ] to add a Java class load breakpoint.

In the resulting dialog:

  • In the Choose a Type field, type a string that is contained in the name of the exception you want to add. You can use wildcards as needed ("* " for any string and "? " for any character).
  • In the Matching items list , select the exception you want to add.
