Searching

Text strings and files can be searched for in the Workbench in two different ways:

  • Using the Search Button and Search view
  • Using the Quick Text Search Dialog

In this section, we discuss both.

First, we start showing how Search can be used to perform a text search across the resources that are shown in the navigation view. Instruction will also be given on how to use the Search view to work with the results.

Second, we will also explain how to use the Quick Text Search as an alternative to the standard search. The Quick Text Search is a popup dialog especially suited to find and jump to different places in your workspace quickly.
