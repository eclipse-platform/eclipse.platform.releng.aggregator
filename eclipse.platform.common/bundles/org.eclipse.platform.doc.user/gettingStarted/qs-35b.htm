Copying

You can copy files to a folder using the following steps.

  1. In one of the navigation views, select the .txt file.

  2. From the file's context menu, select Copy (or Ctrl+C)

  3. In one of the navigation views, select another folder as the destination.

  4. From the folder's context menu, select Paste (or Ctrl+V).

As an alternative to copying files using the copy operation, it is also possible to copy files by holding down the Ctrl key while dragging a file from one folder to another folder. (On macOS, the ‚å• key is used for this.)

Once the file has been copied it can be renamed.
