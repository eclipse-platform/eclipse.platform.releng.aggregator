Renaming

Now that the .txt file has been copied from one folder to another it is ready to be renamed as something else.

  1. In one of the navigation views, select the copied .txt file.

  2. From the file's context menu, select Rename.

  3. The navigation view overlays the file's name with a text field. Type in a new name and press Enter.

    To halt the renaming of a resource, Escape can be pressed to dismiss the text field.

Copy and rename works on folders as well.

  1. In one of the navigation views, select the a folder.

  2. From the folder's context menu choose Rename.

  3. Once again the navigation view overlays the folder name with an entry field to allow the typing in of a new name. Change the folder name to a different one.
  4. Rename the folder back to its original name .
