Editors

You can change the following preferences on the General > Editors preference page:

Option Description Default
Size of recently opened files list Each file that is opened in an editor is stored in a list of recently used files in the File menu. This option controls the number of files that is displayed in that list. 20
Allow in-place system editors Specifies whether or not the workbench may open system editors of OLE applications on Windows in-place. On
Restore editor state at startup Specifies whether or not the workbench should restore the state of open editors on startup. On
Promp to save on close even if still open elsewhere Specifies whether or not the workbench prompts for saving when a dirty editor or view is closed, but the same is still open in other parts On
Close editors automatically Specifies whether or not the workbench should automatically close editors when a given number of editors is reached. On
Number of open editors before closing If Close editors automatically is turned On this specifies the number of editor that are kept open for automatically closing editors. 99
