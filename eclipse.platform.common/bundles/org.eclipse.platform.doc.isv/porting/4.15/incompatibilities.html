Incompatibilities Between Eclipse 4.14 and 4.15

Eclipse changed in incompatible ways between 4.14 and 4.15 in ways that affect plug-ins. The following entries describe the areas that changed and provide instructions for migrating 4.14 plug-ins to 4.15. Note that you only need to look here if you are experiencing problems running your 4.14 plug-in on 4.15.

See also the list of deprecated API removals for this release.
