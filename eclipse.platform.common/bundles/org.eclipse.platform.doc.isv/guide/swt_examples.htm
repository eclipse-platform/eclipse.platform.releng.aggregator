SWT Examples

This page lists places where you can find SWT examples and more information on SWT.

Examples Guide

Within this Platform Plug-in Developer Guide is an Examples Guide that describes how to install and run the Eclipse platform examples. SWT has contributed many of these examples, some as Eclipse plug-ins, and others as standalone SWT examples.

SWT Snippets

A great collection of SWT examples can be found on the SWT Snippets page: http://www.eclipse.org/swt/snippets/. Here, you will find hundreds of simple stand-alone SWT examples.

More Information

For further information on SWT, multi-platform screenshots, FAQ, links to books, articles, and tutorials, and more, please visit: http://www.eclipse.org/swt/.
