Available OSGi Services interfaces to extend and customize the Platform

The Platform can be extended and customized by providing some implementation of OSGi services (for example using Declarative Services).

This document lists the service interfaces extenders can provide to the Platform and that Platform will consume as extensions:

Platform Runtime

  • org.eclipse.core.runtime.IAdapterFactory services contribute adapt strategies, similarly to extensions of org.eclipse.core.runtime.adapter.

Platform UI

  • org.eclipse.e4.ui.workbench.modeling.IModelProcessorContribution services contribute model processors, similarly to extensions of org.eclipse.e4.workbench.model.

Workspace

  • org.eclipse.core.resources.IResourceChangeListener services contribute a resource change listener to the workspace.

Platform Text

  • org.eclipse.jface.text.contentassist.IContentAssistProcessor services contribute to content-assist in the Generic Editor, similarly to extensions of org.eclipse.ui.genericeditor.contentAssistProcessors.
