Help preferences

On the Help preferences page, you can indicate how to display help information.

Option

Description

Default
Open Help Search Configures whether Help/Search (and Help/Index if configured) opens the help view or help contents. In the dynamic help view
Open help view documents This option allows you to determine whether the documents selected in the help view will be opened in-place, in the editor area or in a browser. in-place
Open Help Contents Allows you to open help contents in an external web browser or in the Eclipse help browser. Use "Web Browser" preference page to select browser to use. Off
Open window context help This option allows you to determine whether the window context help will be opened in a dynamic help view or in an infopop. in a dynamic help view
Open dialog context help This option allows you to determine whether the dialog context help will be opened in a dynamic help section of help view or in an infopop. in dialog tray
