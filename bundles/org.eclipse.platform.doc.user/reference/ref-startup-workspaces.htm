Startup and Shutdown, Workspaces

The General > Startup and Shutdown > Workspaces preference page allows configuration of the workspace prompting during IDE startup.

Option
Description
Default
Prompt for workspace on startup
If this option is turned on then the workbench will prompt you each time it is started for what workspace to use.
On
Number of recent workspaces to remember
The maximum number of recently-used workspaces that will be remembered and presented in the "Workspace Launcher" dialog.
5
Recent workspaces
The list of recently used workspaces that are presented in the "Workspace Launcher" dialog.
<none>
