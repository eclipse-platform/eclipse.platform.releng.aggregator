Process Console

The Process Console shows the output of a process and allows you to provide keyboard input to a process.

The Process Console shows three different kinds of text:

  • Standard output
  • Standard error
  • Standard input

You can choose the different colors for these kinds of text on the Run/Debug > Console preference page.

The extra commands available in the Process Console are listed below.

Process Console Commands

Command

Name

Description

Availability

Remove All Terminated Launches Removes all of the terminated launches from the current console. Context menu and view action
Remove Launch Removes the current launch from the console. View action
Terminate Terminates the running launch in the current console. View action
  Select All Selects all of the contents of the current console. Context menu
