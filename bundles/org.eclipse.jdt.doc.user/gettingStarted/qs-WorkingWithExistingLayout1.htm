Sibling products in a common source tree

The source files for the company's products are laid out in one big directory that is version and configuration managed outside Eclipse:

  1. Select the menu item File > New > Project.... to open the New Project wizard.
  2. From the tree display select Java Project. Click Next..
  3. On the next page, type "Product1" in the Project name field. Click Next.

  4. Select "Product1" source folder and click Add Folder....
  5. Click Create New Folder.... Type "JavaSourceFiles" in the Folder name field.
  6. Click Advanced. Check Link to folder in the file system. Click Browse.... and choose the "JavaSourceFiles" directory in "Product1".

  7. Click OK to close the dialog.
  8. Check that "Product1/bin" appears in the Default output folder field.

  9. Click Finish.
  10. Repeat these steps for "Product2".
  11. You now have two Java projects which respectively contain the sources of "Product1" and "Product2".
