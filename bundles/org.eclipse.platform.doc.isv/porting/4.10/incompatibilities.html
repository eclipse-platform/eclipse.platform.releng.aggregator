Incompatibilities between Eclipse 4.9 and 4.10

Eclipse changed in incompatible ways between 4.9 and 4.10 in ways that affect plug-ins. The following entries describe the areas that changed and provide instructions for migrating 4.9 plug-ins to 4.10. Note that you only need to look here if you are experiencing problems running your 4.9 plug-in on 4.10.

See also the list of deprecated API removals for this release.
