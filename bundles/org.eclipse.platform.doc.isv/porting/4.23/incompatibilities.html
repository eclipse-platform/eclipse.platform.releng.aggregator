Incompatibilities between Eclipse 4.22 and 4.23

Eclipse changed in incompatible ways between 4.22 and 4.23 in ways that affect plug-ins. The following entries describe the areas that changed and provide instructions for migrating 4.22 plug-ins to 4.23. Note that you only need to look here if you are experiencing problems running your 4.22 plug-in on 4.23.

See also the list of deprecated API removals for this release.
