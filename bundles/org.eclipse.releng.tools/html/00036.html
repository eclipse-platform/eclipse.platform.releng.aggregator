Test Steps:

  1. Add one file to each of two projects
  2. Commit these two project
  3. Release the two projects
  4. Make sure map file is updated and committed
  5. Selected the two projects
  6. Click Compare With->Released
  7. Ensure there is no change
  8. Edit the two files and save
  9. Add a file to 1st project
  10. Add a folder to 2nd project
  11. Add a file to the folder
  12. Optionally, commit the two projects
  13. Selected the two projects
  14. Click Compare With->Released
  15. Ensure there are two changs for each of the projects
