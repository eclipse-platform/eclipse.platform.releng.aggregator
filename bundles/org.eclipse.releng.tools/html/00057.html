Test Steps:

  1. Select one released project
  2. Make sure there is a corresponding map entry for this project
  3. Delete the project locally
  4. Right click the map file having the entry
  5. Click Team->Load Map Projects

Expected Results:

  1. The project is added to the workspace
  2. A tag name appears to the right side of the project name
  3. The tag is same as the tag for this project in the map file
