Show Map File Change Option on Tag page

Since:
Last Modified: $Date: 2004/03/23 19:11:08 $

Precondition:

  1. Project1 was released as v2 last time
  2. Project2 was released as v1 last time

Test Steps:

  1. Start wizard
  2. Uncheck "Release only changed projects"
  3. Select project1 and project2
  4. Enter "v2" on Tag Page
  5. Check "Move tag if it already exists"
  6. Select "Show the map file change before committing"
  7. Click next
  8. Click back
  9. Modify the tag as "v3"
  10. Click finish

Expected Result:

  1. Step 7: Map compare page will show the difference of entry associated with project2
  2. Step 9: Map compare page will show the difference of entries associated with both project1 and project2
  3. Step 10: Local tags of project1 and project2 are changed to v3, map project is committed and project1 and project2 are released sucessfully
