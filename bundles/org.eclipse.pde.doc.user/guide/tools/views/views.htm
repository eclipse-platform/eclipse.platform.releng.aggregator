Views

PDE provides four views that help plug-in developers inspect different aspects of their development environment. These views can be opened via Window > Show View > Other... > Plug-in Development .

For detailed information on individual PDE views, refer to the following documents:

  • Error Log
  • Plug-in Registry
  • Plug-ins
  • Plug-in Dependencies
  • Target Platform State

Plug-in Development Perspective

The Plug-in Development perspective is designed to support efficient development and integration of plug-in projects. Its also adds the Plug-ins and Error Log views which are particularly useful.

The perspective can be opened via Window > Open Perspective > Other... > Plug-in Development .
