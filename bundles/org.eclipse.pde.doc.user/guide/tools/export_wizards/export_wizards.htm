Export Wizards

PDE provides wizards that build, package and export plug-ins, fragments, features and products into a deployable format. The PDE export wizards are available under File > Export... > Plug-in Development from the top level menu.

For details on individual export wizards, refer to the following documents:

  • Plug-in Export
  • Feature Export
  • Product Export
  • Target Export
