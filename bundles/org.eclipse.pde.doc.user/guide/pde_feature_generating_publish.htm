Publishing a JAR

PDE does not provide automated publishing support for features.

To publish a feature, you need to place its JAR and the referenced plug-in and fragment JARs on a platform update server. You must also edit the update site manifest file to include the feature, or perform an analogous task in case the update site server provides its own implementation of the update site.

See Platform Installation and Update for a description of the update mechanisms and directory structures expected.
