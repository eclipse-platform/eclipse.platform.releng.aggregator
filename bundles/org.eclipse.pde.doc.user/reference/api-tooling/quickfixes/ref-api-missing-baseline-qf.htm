Missing API Baseline Quickfix

This quickfix can be used to quickly help resolve missing API baseline problems.

This quickfix is available in the Problems view.

The Problems view

  • Open the Problem view.
  • Select the missing API baseline problem.

  • Use Ctrl + 1 to trigger the quickfix from the problem view (or use the context menu as shown above).

Completeing the quickfix will open the Plug-in Development > API Baselines preference page, allowing you to create a new baseline using the API baseline wizard.
