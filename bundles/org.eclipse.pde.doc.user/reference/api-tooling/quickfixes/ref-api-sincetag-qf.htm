Since Tag Quickfix

This quickfix can be used to quickly add snice tags or help correct problems witrh existing since tags.

This quick fix is available in one of two places:

  1. The Problems view.
  2. Editing the resource the problem exists in.

The Problems view

  • Open the Problem view.
  • Select a since tag problem.

  • Use Ctrl + 1 to trigger the quickfix from the problem view (or use the context menu as shown above).

Editing the resource

  • Open the resource the API problem exists in.
  • Use Ctrl + 1 on the corresponding problem.


Once the since tag problem has been corrected the corresponding resource is rebuilt and the problem is removed from the Problems view and any open editors.
