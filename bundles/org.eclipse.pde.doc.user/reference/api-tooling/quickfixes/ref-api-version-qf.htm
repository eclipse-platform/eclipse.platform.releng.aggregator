Version Numbering Quick Fix

This quick fix is used to quickly and easily correct version numbering problems for your plug-in.

This quick fix is available in one of two places:

  1. The Problems view.
  2. The manifest editor for the corresponding manifest file.

The Problems view

  • Open the Problems view.
  • Select a version numbering problem.

  • Use Ctrl + 1 to trigger the quick fix from the Problems view (or use the context menu as shown above).

The manifest editor

  • Open the manifest editor on the manifest with the version problems.
  • Navigate to the MANIFEST.MF tab.
  • Use Ctrl + 1 on the corresponding version problem.
