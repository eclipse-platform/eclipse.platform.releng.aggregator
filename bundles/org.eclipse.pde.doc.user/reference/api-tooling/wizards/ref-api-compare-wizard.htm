API Compare Wizard

The API compare wizard can be used to compare a selected set of selected elements (projects, source folders, package fragments, or compilation units) to a selected API baseline and show those results in the API Tools View


Option

Description

Default

Baseline You can choose any API baseline from the combo box to compare to. Default baseline or empty
Add New Baseline Opens the API Baselines preference page so that a new baseline can be added for use.
