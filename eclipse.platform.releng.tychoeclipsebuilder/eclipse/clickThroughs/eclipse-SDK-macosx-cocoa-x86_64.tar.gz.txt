This version of Eclipse for Cocoa requires Mac OS X 10.5 (Leopard) or
higher. 

Please use BOMArchiveHelper if StuffIt fails to extract the downloaded tar.gz
(https://bugs.eclipse.org/bugs/show_bug.cgi?id=131650).

In order to use the SWT_AWT bridge, OS X JRE version 1.5.0 Release 5 (or
greater) must be used.
